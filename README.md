# SmartPrice Insights – Real Estate Platform with ML Price Prediction

SmartPrice Insights is a full-stack real estate web platform that allows users to buy, sell, and explore properties while also predicting property prices using Machine Learning.

This project combines Web Development and Machine Learning into a single platform.

## Features

- User Registration and Login System
- Property Price Prediction using Machine Learning
- Buy Property Module
- Sell Property Module
- Property Image Upload
- Market Analytics Dashboard
- Property Map Explorer
- Property Database stored in CSV
- Secure Password Hashing Authentication
- REST API using Flask

## Technologies Used

- Python
- Flask
- Machine Learning (Scikit-learn)
- Pandas, NumPy
- HTML, CSS, JavaScript
- Leaflet Maps
- CSV Database

## Project Structure



app.py → Main Flask Application
util.py → ML Model & Data Processing
templates/ → All HTML pages
static/ → CSS, JS, Images, Uploads
server/artifacts/ → ML Model Files
server/*.csv → Dataset


## How to Run


pip install -r requirements.txt
python app.py


Then open:

http://127.0.0.1:5000/
