# SmartPrice Insights – Real Estate Platform with ML Price Prediction

SmartPrice Insights is a full-stack real estate web platform where users can buy, sell, and explore properties while also predicting property prices using Machine Learning.

## Features
- User Login & Registration
- Property Price Prediction (Machine Learning)
- Buy Property Module
- Sell Property Module
- Property Image Upload
- Market Analytics Dashboard
- Property Map Explorer
- Property Database (CSV)
- Secure Authentication

## Technologies Used
- Python
- Flask
- Machine Learning (Scikit-learn)
- Pandas, NumPy
- HTML, CSS, JavaScript
- Leaflet Maps

## How to Run
pip install -r requirements.txt  
python app.py

Then open:
http://127.0.0.1:5000/
